4. 진도 통과시키기 (가장 많이 쓰는 기능)

아이가 한 단계를 해냈을 때, 단계 칸(칩)을 탭 한 번이면 통과됩니다.

1칩을 좌우로 넘겨 현재 단계를 확인 (파란 테두리 = 지금 할 단계).
2해낸 단계를 탭 → 통과(회색).
3이전 단계도 자동 통과 (사다리식).
4잘못 눌렀으면 칩 우측 위 ↩로 취소.

5. 기록 입력 — 시간·스트로크·바퀴

측정이 필요한 단계는 통과할 때 기록도 함께 남길 수 있습니다.

  • 시간/스트로크 — 칸에 숫자를 넣고 통과하면 같이 저장됩니다.
  • 기록 입력 ▾ — 카드 아래를 펼치면 전체 단계를 그리드로 보며 입력.

이 기록들이 쌓여 나중에 “이 아이가 언제부터 빨라졌는지”를 보여줍니다.

6. 마스터반 학생 (영법별 바퀴)

네 영법(자유형·배영·평영·접영)을 모두 뗀 아이는 마스터반으로 자동 전환됩니다.

  • 영법별 바퀴 카운터 표시 (1바퀴 = 50m).
  • + / − 로 오늘 바퀴를 올리고 내립니다.
  • 빠르게 눌러도 로딩 없이 즉시 반영 → 자동 저장.
  • IM(개인혼영)도 같은 방식.

7. 학생 대시보드 — 성장 한눈에 + 부모 피드백

오늘 수업에서 아이 이름을 누르면 들어갑니다. 부모 상담·월간 피드백에 그대로 활용됩니다.

  • 프로필 & 현재 단계 — 반·담당 강사·입문일
  • 출석 미니 달력 — 출석/결석 한눈에
  • 영법별 성취 레이더 + 거리·출석률 지표
  • 부모 피드백 초안 — 상담 때 보낼 내용을 미리 메모
  • 일별 활동 — 최근 30일

9. 보강 학생 처리

오늘 수업 화면 맨 위 “오늘의 보강”에서 관리합니다.

  • 보강 온 아이를 검색·추가하면 오늘 내 목록에 들어옵니다.
  • 다른 반 아이라도 내 반으로/가져오기로 담아 기록할 수 있습니다.
  • 기록 방법은 일반 수업과 똑같습니다.
어제·그제도 수정 가능
오늘 화면을 내리면 어제·그제 수업이 있어, 깜빡한 기록을 이틀 전까지 보완할 수 있습니다.

10. 자주 묻는 질문

Q. 기록을 잘못했어요.

통과는 칩의 ↩, 바퀴는 − 버튼으로 바로 되돌릴 수 있습니다.

Q. 출석을 따로 눌러야 하나요?

아니요. 진도나 기록을 남기면 출석은 자동 처리됩니다. 결석만 눌러주세요.

Q. 휴대폰에서만 되나요?

PC·태블릿·휴대폰 모두 됩니다. 현장에서는 휴대폰이 가장 편합니다.

Q. 데이터가 사라질까 걱정돼요.

모든 기록은 즉시 서버에 안전하게 저장됩니다. 기기를 바꿔도 로그인하면 그대로 있습니다.
